How it works

1
Input User uploads an audio file or records voice in the browser.
2
Conversion FFmpeg converts the audio into a format accepted by Speech-to-Text.
3
Analysis Google Speech-to-Text transcribes speech and identifies the language.
4
Storage Firestore stores the transcript, detected language, and summary.
